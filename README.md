# 수정
웹캠에서 화면 오른쪽 영역(사각지대)에 움직이는 물체가 들어오면 '사각지대 주의!' 메시지 표시하기

- 사이드미러 카메라 시스템 → 웹캠 1대
- OpenCV 객체 추적 (KCF, CSRT) → OpenCV 움직임 감지
- 차량 후방 측면 영역 → 화면 오른쪽 고정 영역
- 거리/접근 속도 계산 → 제외 (너무 복잡)
- 위험도 시각화 → 단순 경고 메시지
- 자율주행 연동 → 제외 (하드웨어 없음)
-------------------------------------------

- 움직임 감지 시 "사각지대 주의!"표시
